### What This Dashboard Does
It is an interactive Excel tool that instantly shows **median salaries** for data roles based on real job postings.  
Perfect for:

- **Recruiters & Hiring Managers** – set competitive, market-accurate salaries in seconds  
- **Talent Acquisition teams** – benchmark offers across titles, countries, and contract types  
- **Compensation & People Ops** – build data-driven salary bands  
- **Job seekers** – know your worth before the offer stage

### Salary Calculator (Sheet: Salary_Calculator)
Just pick 3 things → get the exact median salary:  
1. **Job Title** (dropdown) – Data Analyst, Data Scientist, Data Engineer, Senior roles, ML Engineer, etc.  
2. **Country** (110+ countries)  
3. **Employment Type** (Full-time / Contractor / Part-time / Internship)  

Result → Median annual salary in USD
